
# Weyve 

Weyve is a travel-focused social media platform that enables travelers to share experiences, stories, photos, and videos while connecting with like-minded explorers. The platform combines social networking and community-driven discussions, allowing users to create posts, engage through likes and comments, follow other travelers, join travel communities, and discover destinations through personalized content feeds.

Built using Flutter, Spring Boot, and PostgreSQL, Weyve focuses on providing a scalable, secure, and engaging environment for travelers to document journeys, exchange insights, and build meaningful connections around shared travel interests.
